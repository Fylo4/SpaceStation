Blueprints

I'm a professional web developer. If you are looking to make your own website and don't spend 8 hours a day working with these technologies, there are definitely simpler ways to do it.

Tech Stack

My goal was to use tech I already know how to use and keep it as simple and cheap as possible.

  • Frontend - Angular. Angular Material is used, but pretty much only on the Comment component. This will likely be removed in the future because I don't like relying on Material.
  • Hosting - Netlify. It's free, easy, and is automated with Git pushes.
  • Backend - Netlify Functions. I didn't know Netlify had this until I started on the comment component. It's also pretty easy to use, and since I'm already using Netlify, is one less tech I have to worry about.
  • Database - CockroachDB. Never used it before, but it seems cheap and has everything I need for my extremely minimal DB needs.
  • Version control - GitHub.
